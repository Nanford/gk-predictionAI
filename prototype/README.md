# 晨光志愿 AI 交互原型

高考志愿 AI 预测系统 1.0 可点击交互原型。当前完整演示广东省普通类物理方向，包含院校预测、专业预测、选科校验、AI 辅助解释、志愿单诊断和 AI 使用费用说明。

## 启动

```bash
npm install
npm run dev
```

打开：

```text
http://127.0.0.1:5173
```

## 验证

```bash
npm test
npm run build
```

## 原型边界

- 使用本地演示数据，不代表真实录取数据。
- AI 分析为本地异步模拟，不接入真实 DeepSeek 密钥。
- 支付为模拟确认，不接入真实支付渠道。
- 预测结果仅供辅助参考，不代表最终录取结果。
